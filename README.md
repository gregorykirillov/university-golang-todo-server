# university-golang-todo-server

## .env file:
- ### `DB_CONN_STRING=postgresql://[userspec@][hostspec][/dbname][?paramspec]`

## Run application:
- ### `go run main.go`

## Address:
- ### http://localhost:5000

## Routes
### GET:
- ### /tasks
- ### /task/{id}

### POST:
- ### /addTask

### PUT:
- ### /updateTask/{id}

### DELETE:
- ### /deleteTask/{id}
